# Code of Conduct

This project has adopted the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

All contributors, maintainers, and participants are expected to abide by its terms.

## Reporting

Instances of unacceptable behavior may be reported to the project maintainers at **support@rtrvr.ai**.

All complaints will be reviewed and investigated promptly and fairly. Maintainers are obligated to respect the privacy and security of the reporter.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
