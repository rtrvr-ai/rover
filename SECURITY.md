# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Rover, please report it responsibly.

**Email:** [support@rtrvr.ai](mailto:support@rtrvr.ai)

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

## Response Timeline

- **Acknowledgment:** within 48 hours
- **Initial assessment:** within 5 business days
- **Critical patches:** within 30 days

## Scope

This policy covers the Rover SDK and all packages in this repository. For vulnerabilities in the hosted backend service, please also contact [support@rtrvr.ai](mailto:support@rtrvr.ai).

## Disclosure

We ask that you do not publicly disclose the vulnerability until we have had a chance to address it. We will coordinate with you on disclosure timing.
